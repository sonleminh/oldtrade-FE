<h1>Old Trade</h1>

<h3>Description</h3>
Clone chotot.com website using TypeScript with React, MongoDB, Chakra UI, Socket.IO

<h4>Features:</h4>

<p>- Sign up, login, verify account</p>
<p>- Post, edit, delete post</p>
<p>- Chat with other people</p>
